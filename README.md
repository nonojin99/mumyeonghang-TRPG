# 무명행 TextRPG — 프로토타입 (화산 오리진 수직 슬라이스)

카드게임 「무명행」(`../tower_card_v14.html`)과 **같은 세계, 별개 프로젝트**.
이 폴더는 카드게임과 완전히 격리되어 있다 — 공용 자산(정본·설정집)은 `../`에서 참조만 하고, 카드게임 코드/하네스/`schedule.json`은 건드리지 않는다.

## 구성

```
textrpg/
├─ index.html        엔진 (단일 HTML, 서버 불필요 — 더블클릭으로 실행)
├─ data/
│  ├─ origins.js     2층: 진영 줄기(FACTIONS) + 문파 변주(ORIGINS) + 무공 풀(ARTS_POOL)
│  └─ scenes.js      장면 그래프 (기승전결 1챕터, 10장면·3엔딩)
├─ lint.js           그래프 린터 (카드게임 regress.js의 서사판, 의존성 없음)
└─ README.md
```

## 플레이

`index.html`을 브라우저로 연다(서버 불필요). 화산 오리진 → 기(무명) → 승(입문·무공추첨) → 전(위기, 시간제한 선택) → 결(엔딩). 엔딩 화면에 행로 플로우차트 + 탐색%·발견 엔딩이 누적 표시된다.

## 구현된 기획서 요소

- **3계층 오리진 구조**(기획서 §3): 서사는 진영(`FACTIONS`) 줄기, 문파는 데이터 변주(`ORIGINS`). 타이틀의 잠금 카드(흑도·빙궁·혈교)가 "데이터만 추가하면 붙는" 구조를 보여준다.
- **기승전결 골격**(§2)과 **수렴점**: 3갈래 도입(공손/도발/비웃음)이 위기(`crisis_ambush`)에서 수렴 후 다시 분기.
- **분기 3종**(§4): 즉시 분기 / 지연 씨앗(놓친 살수 → 협객 엔딩의 후환 문구) / 수렴.
- **무공 = 선택지 게이트**: 입문 시 등급 가중 추첨(`rollMartialArt`, maxTier 3). tier≥2(낙영/매화) 보유 시 위기에서 최선의 선택지(❖)가 열린다.
- **시간제한 선택**(§4): 위기 장면 7초 게이지, 초과 시 페널티(둘 다 잃음 + 오명).
- **디제틱 상태창**: HP 수치 대신 俠名/惡名/무공/내상.

## 검증

```bash
node lint.js          # 그래프 무결성 (goto·고아·소프트락·엔딩 도달·플래그 위생·타임드 폴백)
```
현재: 장면 10 · 엔딩 3 · 도달 10/10 · 오류 0 · 경고 0.
전 경로 완전 탐색(117 경로) 결과 막다른 길 0, 3종 엔딩 모두 도달.

## 오리진 추가하는 법 (확장)

1. `data/origins.js`의 `ORIGINS`에 문파 1개 추가(faction·place·startScene), 필요시 `ARTS_POOL`에 무공 풀.
2. `data/scenes.js`에 그 오리진의 도입 장면들 추가(수렴점 `crisis_ambush`는 공유 가능).
3. `node lint.js` 통과 확인.

진영 줄기가 다르면(사파·새외 등) 위기/엔딩의 서사 색만 그 줄기에 맞게 변주하면 된다.
